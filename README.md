# easy_anno
easy_anno is a tool designed to make creating and saving map guide annotations less of a hassle. Written in Counter-Strike 2's new JavaScript-based API, easy_anno provides a set of custom chat commands which simplify the process of creating annotations, automating repetitive tasks by taking advantage of game event hooks and managing internal state.
